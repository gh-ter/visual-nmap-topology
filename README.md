# NetGraph — Visual Network Topology UI
<img width="2560" height="2048" alt="PNG image" src="https://github.com/user-attachments/assets/4ecdbd7d-0c06-450e-8e99-2ca500b7329d" />

NetGraph — это визуальный интерфейс для отображения сетевой топологии на основе результатов сканирования. Проект показывает устройства, соединения, порты и сервисы в виде интерактивного графа с окнами-узлами и линиями связей. Основной упор сделан на наглядность, минималистичный UI и расширяемую архитектуру.

Проект сейчас находится на стадии концепта и UI-прототипа.

##  Возможности

- Визуализация сети в виде графа
- Узлы-окна с краткой информацией об устройствах
- Отображение соединений и направлений трафика
- Подсветка активного сканирующего хоста
- Drag & drop перемещение узлов
- Минималистичный “terminal / retro” стиль интерфейса
- Поддержка иконок и статусных индикаторов
- Браузерный GUI (HTML + Tailwind + SVG)

## Технологии

- HTML5
- TailwindCSS
- SVG для линий соединений
- JavaScript (drag, анимации, интерактив)
- Material Icons / Symbols
